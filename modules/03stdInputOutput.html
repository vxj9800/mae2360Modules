A Note on Using Streams

The header file iostream allows for the use of cout and cin (among others).

While in C the standard for input/output is printf and scanf, in C++ the standard for input/output is often cout and cin.

The cout and cin commands are not compatible with C code, they are exclusive to C++.

What this means to first time programmers: You need to use the g++ compiler instead of gcc!
