Comparing Strings

Strings cannot be compared with a simple a == b command; Instead, the function strcmp(str1,str2) must be used. The strcmp() function reads the two strings, one character at a time, until either different characters or the NULL character is encountered. If the NULL is encountered, then the strings are the same and the function returns a 0. If different characters are encountered, then the ASCII position of str1 relative to str2 is returned.
